# Material Theme Biulder

This is a front-end webpage for building Material Design 3 CSS Tokens.

Website: https://material-theme-builder.sandlada.com/

Support for selecting variants:

- Monochrome
- Neutral
- TonalSpot
- Vibrant
- Expressive
- Fidelity
- Content
- Rainbow
- FruitSalad

Supported current spec version:

- 2021
- 2025

Supports contrast level:

- -1
- 0
- 1
